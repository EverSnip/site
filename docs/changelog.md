# 更新日志

本页面记录了拾光忆栈的所有版本更新内容。

## v1.0.0

**发布日期**：2023年12月15日

### 新功能

- 首次正式发布
- 支持从网页保存文本、图片和视频
- 基本的分类和标签管理
- 全文搜索功能
- 多种视图模式（列表、卡片、杂志）
- 基本的云存储同步（支持华为云OBS）
- 用户设置和偏好管理
- 暗色模式支持
- 基本的导入导出功能

### 已知问题

- 在某些网站上视频保存可能不稳定
- 大量记忆时搜索性能可能下降
- 某些复杂格式的富文本可能无法完全保留
